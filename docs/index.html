XIV Gear Lab

A Windows-first FFXIV gear recommendation and optimisation tool under active development. This public page exists to support testing and signed data updates. It is not a finished release and should not be treated as authoritative gearing advice.

Current state

  • Level-100 current-tier support for the 21 standard combat jobs.
  • Transparent single-action comparison proxy, not rotation or encounter DPS.
  • Current Savage and tomestone equipment only; historical and broader acquisition data are incomplete.
  • Interfaces, formulas and stored-data formats may still change.

Project boundary

Private/non-commercial prototype work has moved into a public testing phase. There is no support promise, uptime guarantee or recommendation that normal players rely on this build yet.
